# About
rook.fm is a platform that connects artists to fans and provides access to a vast music library on the official Montaime web radio

# System Requirements
> [!NOTE]
> This software has not been released under an OSI license. It is not intended for self-hosting!

- Redis
- Pusher-compatible websocket server for Laravel broadcasting
- Azuracast for the radio and on-demand music library
- Standard Laravel requirements
